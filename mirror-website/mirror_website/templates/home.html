Welcome to Mirror

This site is a public repository of Free and Open Source Software hosted by the Clarkson Open Source Institute at Clarkson University in Potsdam, NY, US. This service strives to provide exposure to and ensure availability of open source software and operating systems to the general public. We mirror many Linux distributions, several third party software repositories, and miscellaneous data dumps.

Distributions

We host over 20 distributions, including Debian, Ubuntu, Arch Linux, Linux Mint, and many more. Many are official mirrors, and the majority are updated 4 times per day.

Software

We host a variety of third party software including Blender, KiCad, Cygwin, Eclipse, among many others.

Miscellaneous

We also host miscellaneous projects, including Wikipedia dumps and torrents.

Contact Us

To get in touch with the Mirror team, send a message to mirroradmin [AT] clarkson [DOT] edu We check this inbox periodically, however if a matter is time-sensitive we ask that you also contact COSI directly.

Request a Mirror

We are always looking for new projects to mirror! If there's something you'd like us to mirror, please open an issue with the "Mirror Request" tag on our GitHub repository. The issue template includes instructions on what information we need to start syncing the mirror. If possible, rsync is preferred, but we can also sync over HTTP.
