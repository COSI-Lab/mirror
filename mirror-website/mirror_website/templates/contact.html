Contact Us

To get in touch with the Mirror team, send a message to mirroradmin[@]clarkson[.]edu. We check this inbox periodically, however if a matter is time-sensitive we ask that you also contact COSI directly.

Request a Mirror

We are always looking for new projects to mirror! If there's something you'd like us to mirror, please open an issue with the "Mirror Request" tag on our Github repository. The issue template includes instructions on what information we need to start syncing the mirror. If possible, rsync is preferred, but we can also sync over HTTP.
